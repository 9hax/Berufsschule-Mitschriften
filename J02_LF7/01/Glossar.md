# Glossar

| Begriff | Definition |
| ------- | ---------- |
| Smart-Home-Segment | Ein Smart-Home-Segment unterteilt Smart-Home-Elektronik in verschiedene Bereiche, die bestimmte Ziele verfolgen. Meist werden die Bereiche "Gebäudesicherheit", "Lichtsteuerung", "Energie- und Heizungssteuerung", "Hausgeräte", "Entertainment" und "Beschattung" voneinander getrennt. |
| Gebäudesicherheit | Diese Smart-Home-Sparte beinhaltet Einrichtungen wie Alarmanlagen, Tür- und Fenstersensoren sowie Zutrittskontrollsysteme zum Schutz von Personen und Eigentum. |
| Lichtsteuerung | Durch Lichtsteuerung können Lampen und Lampengruppen für den jeweiligen Gebrauch automatisiert und zentral gesteuert werden. |
| Energie- und Heizungssteuerung | Die Steuerung von Heizkörpern, Thermostaten und Energiemanagement, um Komfort zu erhöhen und Energieverbrauch zu reduzieren. |
| Hausgeräte | Vernetzte Haushaltsgeräte (z. B. Waschmaschine, Kühlschrank, Backofen), die per App oder Automatisierung gesteuert werden können. |
| Entertainment | Smart-Home-Integration von Audio-, Video- und Multimediageräten zur zentralen Steuerung von Filmschauen, Musik und Medienwiedergabe. |
| Beschattung | Automatisierte Steuerung von Rollläden, Jalousien und Markisen zur Lichtregelung, Sichtschutz und Wärmereduktion. |
| Übertragungsmedium | Ein physisches oder drahtloses Medium (Kabel, WLAN, Zigbee, Z-Wave etc.), über das Smart-Home-Geräte Daten austauschen. |
| Funkstandard | Ein standardisiertes Protokoll für drahtlose Kommunikation im Smart-Home (z. B. Zigbee, Z-Wave, Bluetooth, WLAN). |
| Interoperabilität | Die Fähigkeit verschiedener Smart-Home-Systeme und Geräte, miteinander zu kommunizieren und zusammenzuarbeiten. |
| parametrieren | Das Einstellen und Anpassen von Geräteparametern (Zeitpläne, Schaltschwellen, Szenen) zur individuellen Konfiguration. |
| Geräteverknüpfung | Das Einrichten von Regeln und Automationen, bei denen ein Gerät (z. B. Bewegungssensor) andere Geräte (z. B. Licht) auslöst. |
| Gruppensteuerung | Gemeinsame Steuerung mehrerer Geräte als Gruppe (z. B. alle Lampen im Wohnzimmer zusammen ein-/ausschalten). |
| Zentralverknüpfung | Eine zentrale Automationslogik im Smart-Home-Hub, die mehrere Geräte übergreifend verbindet und steuert. |
| Smart-Home-Zentrale | Zentrale Steuereinheit (Hub/Gateway) eines Smart-Home-Systems, die Geräte verbindet, Automationen ausführt und Benutzerschnittstellen (App/Web) bereitstellt. |
| Heizkörpersteuerung | Automatisierte Regelung von Heizkörpern über Thermostate oder Aktoren, um Raumtemperatur, Zeitpläne und Energieeffizienz zu steuern. |
| Messwert / Sensorwert | Ein quantifizierter Datenpunkt (z. B. Temperatur, Luftfeuchtigkeit, Helligkeit), den ein Sensor im Smart-Home erfasst und an das System übermittelt. |
