## Pläne zeichnen

### Buchstaben

| Buchstabe | Bedeutung |
| - | - |
| A | Verteiler |
| Q | Schalter |
| X | Verbindung (Steckdosen und Unterverteiler) |
| E | Lampen und Wärme |
| F | Schutzeinrichtungen |
| P | Darstellung von Informationen (Zähler, LCD, etc...) |

Mehrere Lampen, die vom gleichen Schalter geschaltet werden, werden mit der Nummer des Schalters ergänzt.  
Beispiel: Lampen E1, E2, E3 von Schalter Q2 werden E1.2, E2.2, E3.2 bezeichnet.
