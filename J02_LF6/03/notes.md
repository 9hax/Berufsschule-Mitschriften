# EPK

Eine EPK beginnt immer mit einem Ereignis (Zustand). Ereignisse sind Sechsecke.

Darauf folgt eine Funktion (Handlung), die durch ein abgerundetes Rechteck angegeben wird.

Anschließend folgt wieder ein Ereignis (Zustand).

Funktionen können durch Orga-Einheiten durchgeführt werden. Dazu wird ein nicht-abgerundetes Rechteck genutzt, wessen Pfeil auf die Funktion zeigt.

Ebenso können Systeme zur Ausführung verwendet werden, die mit einem Doppelpfeil verbunden werden. 
Wenn es einen Input oder einen Output gibt, werden die Pfeile in Richtung des Informationsflusses gesetzt.
